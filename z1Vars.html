Variables

Every variable name in Column starts with '#' :
#MyVariable

Variable can have sub variables :
#MyVariable|SubVariable

To Assign variable some value use '::' :
#MyVariable::10;
#MyVariable|SubVariable::"Hello, Column!";

To Get Pointer to the variable use ':' :
:#MyVariable;

You can also store variable pointer in variable :
#OtherVariable:: :#MyVariable;

To get value of the pointer use '=' :
#OtherVariable:: =MyVariable;

Moreover you can make variable act as a variable in pointer :
:OtherVariable :: :MyVariable;
